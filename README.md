# kafka-distributed-testing-framework
A distributed systems testing framework designed to validate event-driven microservices using Apache Kafka. The framework automates UI, API, and Kafka event validation to ensure reliability of asynchronous messaging workflows in distributed architectures.


This project demonstrates how to test event-driven microservices architectures using Apache Kafka.

The framework validates:
• UI workflows
• REST APIs
• Kafka event streams
• asynchronous producer-consumer messaging
• distributed service communication

Tech Stack:
Playwright
Spring Boot
Apache Kafka
Docker
REST Assured
